# Heart-disease-Machine-Learning
This Project focuses on predicting heart disease using machine learning techniques. It implements a complete end -to-end pipeline including data preprocessing, feature scaling, model training, and performance evaluation.
The objective is to build a reliable classification model that can assist in early detection of heart disease based on clinical features. Logistic Regression and Random Forest algorithms are used to analyze and compare performance.

This project demonstrates practical application of machine learning in healthcare and serves as a foundational step towards developing intelligent medical decision support systems.
